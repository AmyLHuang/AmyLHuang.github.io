## About

MyPortfolio is a web application designed to showcase my projects in a clean, professional format. Built using pure HTML, CSS, and JavaScript, it emphasizes simplicity and functionality without reliance on heavy frameworks.s

## Tech Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages (hosting)
- VS Code (IDE)

## Additional Features

- Clean project showcase with image previews and descriptions
- Responsive design for mobile and desktop
- Lightweight, fast-loading pages
- Particle Effects that follow mouse movement
