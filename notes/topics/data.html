Data Analytics

Data Analytics is the process of examing raw data, in the form of datasets, in order to draw conclusions about the info they contain. Some common tools used by data analysts are: Microsoft Excel, Google Sheets, Tableau, Microsoft Power BI, SQL, Python, and R.

Data

Data is a collection of facts, figures, and observations presented in a structured or unstructured format. It provides the raw material for informed decision-making, innovation, and problem-solving.

Data --> Knowledge --> Action.

Data quality matter! Poor data quality can lead to inaccurate insights, flawed conclusions, and poor business decisions. Some core quality-checks to evaluate the quality of the data collected are:

  1. Accuracy: Ensure the data is correct and free of errors.
  2. Completeness: Having a complete dataset without missing values.
  3. Consistency: Maintaining uniformity in data formats and definitions.
  4. Relevance: Collecting data that is pertinent to the specific analysis or project.

Excel

Lock formulas
When we copy and paste/autofill formulas to other cells, the cell references can change, which may sometimes cause errors in our calculations. By locking formulas using the $ symbol, it will prevent the cell reference from changing. Shortcut is F4.

VLOOKUP (aka Vertical Lookup)
Syntax: =VLOOKUP(lookup_value, table_array, col_index_num, [is_exact_match])

  • useful in finding and extracting info related to a certain identifier/look up value quickly
  • can be used to combine the data from various sources based on a shared key
  • col_index_num starts at 1
  • can only look for values to the right of the key column
  • only finds first match

Index match
Syntax: =INDEX(result_array, MATCH(lookup_value, lookup_array, [match_type]))

  • can look up values in columns to the left of the lookup column (unlike VLOOKUP)
  • The MATCH() function is used to find the row number where the lookup value is present in the lookup array. The INDEX() function then returns the value in the array at that row number.

Pivot Table
A pivot table is an interative function (not a formula) that collects all your data and quickly summarizes it for you to organize your info more efficiently. This summary might include sums, averages, or other statistics, which the pivot table groups together using a chosen aggregation function applied to the grouped value.

Exploratory Data Analysis (EDA)

EDA is the process of investigating, organizing, and analyzing datasets and summarizing their main characteristics, often employing data wrangling and visualization methods. It is not a step-by-step process, but rather an iterative and non-sequential process.

The six practices of EDA are:

  • Discovering: Familiarize with the data
  • Structuring: Take raw data and organize/transform it to be more easily visualized, explained, or modeled
  • Cleaning: Remove errors that might distort the data or make is less useful
  • Joining: Augument data by adding values from other datasets
  • Validating: Verify that the data is consistent and high quality
  • Presenting: Make a cleaned dataset available to others for analysis or further modeling

PACE Workflow

A workflow to help focus on the end goal of any given dataset. It stands for plan, analyze, construct, and execute.
