NumPy

import numpy as np

Vectorization enables operations to be performed on multiple components of a data object at the same time.

n-dimensional Arrays

Ndarrays are the core structure in NumPy. Elements can be changed by indexing and replacing values, but arrays cannot be lengthened or shortened since they allocate a contiguous memory block.

Initialization
  • np.array(list) — creates an ndarray
  • np.zeros((dim1, dim2, ...)) — array filled with zeros
  • np.ones((dim1, dim2, ...)) — array filled with ones
  • np.full((dim1, dim2, ...), value) — array filled with a specific value
  • np.min(ndarray), np.max(ndarray), np.median(ndarray), np.std(ndarray), np.mean(ndarray)
Methods
  • ndarray.flatten()
  • ndarray.reshape(dim1, dim2, ...)
  • ndarray.tolist()
Attributes
  • ndarray.shape — dimensions of the array
  • ndarray.dtype — data type
  • ndarray.size — total number of elements
  • ndarray.T — transposed array

pandas

import pandas as pd

pandas has two core objects: DataFrame (A 2D labeled data structure with rows and columns) and Series (A 1D labeled array).

Methods
  • pd.read_csv(source) — load a CSV file
  • dataframe.head(num_rows)
  • dataframe[col].mean(), dataframe[col].max(), ...
  • dataframe.describe()
  • dataframe.info()
  • dataframe.value_counts()
  • dataframe[conditions] — aka Boolean Masking
  • dataframe.iloc[row_idx, col_idx]
  • dataframe.loc[row_selector(s), col_name(s)]
  • dataframe[col_name], dataframe[[col1, col2, ...]]
Attributes
  • dataframe.columns
  • dataframe.shape
  • dataframe.info
Grouping and Aggregation
  • groupby(): A DataFrame method that gr oups rows of the dataframe together based on their values at one or more columns, which allows further analysis of the groups. For example: dataframe.groupby([col1, col2, ...]).sum(numeric_only=True)
  • agg(): A groupby method that allows you to apply multiple calculations to groups of data. For example: dataframe.groupby([col1, col2, ...]).agg(['mean', 'median'])
Merging and Joining Data
  • pd.merge(df1, df2, on='key', how=['inner'|'outer'|'left'|'right']): A pandas function that joins two dataframes together; it only combines data by extending along axis one horizonally.
  • pd.concat([df1, df2, ...], axis=[0|1]): A pandas function that combines data either by adding it horizontally as new columsn for existing rows, or vertically as new rows for existing columns. Use the df.reset_index(drop=True) method to reset the indices.
