Google Advanced Data Analytics (Coursera)

Automatidata

Project goal: In this fictional scenario, the New York City Taxi and Limousine Commission (TLC) has approached the data consulting firm Automatidata to develop an app that enables TLC riders to estimate taxi fares in advance of their ride.

Background: Since 1971, TLC has been regulating and overseeing the licensing of New York City's taxi cabs, for-hire vehicles, commuter vans, and paratransit vehicles.

Scenario: The recently submitted New York City TLC project proposal has been approved. The Automatidata team now has access to the TLC data to analyze, identify key variables, and prepare for exploratory data analysis.

Course 2 tasks:

  • Load, explore, and extract the TLC data with Python
  • Use custom functions to organize the dataset
  • Build a dataframe for the TLC project
  • Create an executive summary for Automatidata

Note: This story is fictitious. The data has been created for pedagogical purposes.

TikTok

Project goal: The TikTok data team is developing a machine learning model for classifying claims made in videos.

Background: TikTok is the leading destination for short-form mobile video. Its mission is to create a place for inclusive, joyful, and authentic content.

Scenario: As a data analyst on TikTok's data team, you’ll prepare the claims dataset, organize it for exploratory data analysis, and share progress insights.

Course 2 tasks:

  • Build a dataframe for the TikTok dataset
  • Read data from a CSV file
  • Display rows and examine data types
  • Gather descriptive statistics
  • Visualize the data in Python
  • Report to TikTok’s data team through an executive summary

Note: This story is fictitious. The data has been created for pedagogical purposes.

Waze

Project goal: Waze leadership asked the data team to develop a machine learning model to predict monthly user churn. An accurate model helps prevent churn, improve retention, and grow Waze’s business.

Background: Waze’s free navigation app is supported by a community of editors, testers, translators, partners, and users.

Scenario: The churn project has been approved and the team has access to user data. Before analysis, the data must be inspected, organized, and prepared.

Course 2 tasks:

  • Import user data
  • Create and inspect a dataframe
  • Identify outliers
  • Create data visualizations
  • Share an executive summary with the Waze data team

Note: This story is fictitious. The data has been created for pedagogical purposes.
