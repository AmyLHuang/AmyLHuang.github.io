Certificates

Google Advanced Data Analytics Professional Certificate (Coursera), 2025

This certificate from Google taught me a bunch of practical, real‑world data analysis skills that go beyond the basics. Over the 8‑course series, I learned how to explore and clean messy datasets, make clear visualizations, build statistical and machine learning models, and pull out insights that are useful for decision‑making. The program covers Python programming, regression and predictive modeling, data visualization tools like Tableau, and how to communicate results clearly to stakeholders — all through hands‑on projects and practice exercises. I also completed a capstone project where I put everything together by examining real data and building models to find trends and patterns.

SAS Programming 1: Essentials (SAS.com), 2025

I completed SAS’s Programming 1: Essentials course, which taught me how to write and run programs using the SAS language. I learned how to navigate the SAS programming environment, read data from different sources (including raw data and Excel), clean and validate datasets, create and manipulate variables, and combine data tables. The course also covered how to produce useful reports and summaries, giving me a solid understanding of how to work with data in SAS from start to finish.
