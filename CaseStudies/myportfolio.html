Introduction

MyPortfolio is a web application designed to showcase my projects in a clean, professional format. Built using pure HTML, CSS, and JavaScript, it emphasizes simplicity and functionality without reliance on heavy frameworks.

Goal

Develop a self-contained, customizable portfolio website using core web technologies, allowing full creative control, fast load times, and easy hosting on platforms like GitHub Pages.

Key Features

  • Clean project showcase with image previews and descriptions
  • Responsive design for mobile and desktop
  • Lightweight, fast-loading pages
  • Particle Effects that follow mouse movement

Technologies Used

HTML5, CSS3, JavaScript (DOM manipulation), GitHub Pages (hosting), VS Code (IDE)
