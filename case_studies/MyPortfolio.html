Introduction

MyPortfolio is a web application designed to help users showcase their work experience and projects in a clean, professional format. Built using pure HTML, CSS, and JavaScript, it emphasizes simplicity and functionality without reliance on heavy frameworks.

Goal

Develop a self-contained, customizable portfolio website using core web technologies, allowing full creative control, fast load times, and easy hosting on platforms like GitHub Pages.

Key Features

  • Clean project showcase with image previews and descriptions
  • Responsive design for mobile and desktop
  • Lightweight, fast-loading pages

Technologies Used

HTML5, CSS3 (Flexbox, Grid), JavaScript (DOM manipulation), GitHub Pages (hosting)
